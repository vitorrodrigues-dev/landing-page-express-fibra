# Aviso de Uso e Direitos

Este projeto foi desenvolvido por Vitor Rodrigues para fins de estudo, prática profissional e composição de portfólio.

A aplicação foi criada para uma empresa real, utilizando identidade visual, informações institucionais e elementos comerciais autorizados exclusivamente para demonstração pública do trabalho desenvolvido.

Apesar do repositório estar disponível publicamente para visualização e aprendizado, não é permitida a reprodução, redistribuição, revenda, reutilização comercial ou cópia integral/parcial deste projeto sem autorização prévia.

O código-fonte pode servir como referência educacional para outros desenvolvedores, mas os elementos relacionados à marca, identidade visual, conteúdos institucionais, imagens e contexto comercial pertencem aos seus respectivos responsáveis.

Este projeto não possui finalidade open source.
